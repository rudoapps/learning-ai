👁 RAG vs 🎓 Fine-tuning vs 📋 Prompting

Tres formas de enseñarle algo a la IA. No son alternativas — a veces se combinan.

📋 Prompting 👁 RAG 🎓 Fine-tuning
Qué es Instrucciones en el prompt Buscar en tus docs y pegarlos al prompt Re-entrenar parcialmente los pesos del modelo
Coste setup $0 — solo escribir Bajo — indexar docs + base vectorial Alto — dataset + horas de GPU ($200+)
Tiempo setup Minutos Horas Días
Actualizable Inmediato (editas el prompt) Inmediato (cambias el doc, se reindexa) Lento (hay que re-entrenar)
Escala No — limitado por context window Sí — millones de documentos No — para estilo, no datos
Enseña hechos Sí (pocos, en el prompt) Sí (muchos, con citas) No (solo estilo/formato)
Enseña estilo Parcial (instrucciones) No (solo datos) Sí (interioriza patrones)
Cita fuentes No Sí No
Coste por turno Bajo (solo el prompt) Medio (prompt + chunks) Bajo (prompt corto) pero modelo más caro

🌳 Árbol de decisión

❓ ¿Quieres que la IA sepa cosas (hechos, documentos) o que hable de cierta forma (tono, formato)?
📚 Saber cosas → ¿cuántos documentos?
Pocos (< 10 páginas) → Prompting (mételos en el prompt directamente)
Muchos (100+) → RAG (indexa y busca dinámicamente)
🎨 Hablar de cierta forma → ¿puedes describirlo con palabras?
Sí → Prompting (instrucciones detalladas + few-shot)
No, es un patrón sutil → Fine-tuning (miles de ejemplos)
⚠️ ¿Quieres AMBAS cosas? → RAG + Prompting (lo más común en producción)

🏢 Ejemplo: bufete de abogados

Necesidad 1: responder sobre contratos del bufete

→ RAG. Indexas 2.000 contratos, el asistente busca los relevantes por cada consulta y cita fuentes.

Necesidad 2: tono legal formal español

→ Prompting primero ("usa tono formal, cita artículos del CC"). Si no basta → Fine-tuning con 5.000 ejemplos de respuestas ideales.

Necesidad 3: reglas fijas del bufete

→ Prompting. "Nunca prometas resultados judiciales. Siempre recomienda consulta presencial para casos penales."

En producción real: RAG + prompting afinados cubren el 90% de los casos. El fine-tuning es la herramienta menos usada y más sobrevalorada — solo para estilo muy específico que no se puede describir con instrucciones.
